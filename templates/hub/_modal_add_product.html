{% comment %} Body for the product modal. Handles BOTH create and edit modes. The outer modal state lives on the `guildCart` Alpine component: - productModalMode: 'create' | 'edit' - productModalInitial: { pk, name, price, action, splits: [{recipient_type, guild, percent}] } When mode === 'create', the form posts to hub_guild_product_create and the splits start at the default admin/guild split. When mode === 'edit', the form posts to hub_guild_product_update and splits load from the product. The splits formset field names match billing.forms.build_product_split_formset() with prefix="splits": each row submits splits-N-recipient_type, splits-N-guild, splits-N-percent. {% endcomment %}
